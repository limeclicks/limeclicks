Common Tasks

Add Keywords
  1. Click Add keywords (top-right)
  2. Paste or type one keyword per line
  3. Pick Country and optional Tags
  4. Save to start tracking
Find "Quick Wins"
  1. Filter Country as needed
  2. Sort by Rank or filter ranks 11-20
  3. Target these pages with on-page improvements
  4. Add internal links to boost rankings
Monitor Declines
  1. Click Declined on filter pills
  2. Sort by Change to see biggest drops
  3. Inspect the listed URL
  4. Ensure correct page is ranking
Triage "Not Ranking"
  1. Click Not Ranking filter
  2. Check if relevant page exists
  3. Verify page is indexable
  4. Create/optimize target page
  5. Add internal links

Best Practices

  • Tag everything (location, service type, funnel stage)
  • Group by intent and optimize pages accordingly
  • Watch URL mismatches and strengthen internal links
  • Track plurals, synonyms, and near-variants
